## How to use this repository

### If you are learning alone
Read the roadmap first, then move through the lessons in order.

### If you are teaching
Read the teacher guide first, then use each lesson as your classroom note.

### If you want practice
Use the assessment and assignment section at the end of each lesson.

## Teaching design

Each lesson is structured to include:

- objective
- explanation
- example
- key terms
- common mistakes
- assessment
- assignment
- instructor guidance where needed

## Learning design

This handbook follows a simple rule:

Start from simple ideas, then move into real Spring Boot work gradually.
