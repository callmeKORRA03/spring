# Spring Boot Interview Q and A

This file contains beginner to junior level Spring Boot interview questions and simple answer ideas.

## 1. What is Spring Boot?

Spring Boot is a tool built on top of Spring that helps developers build Java applications faster with less manual setup.

## 2. What is the difference between Spring and Spring Boot?

Spring is the larger framework.
Spring Boot is the easier setup focused way of building Spring based applications.

## 3. What is `@RestController` used for?

It is used for controllers that return data directly, such as text or JSON.

## 4. What is the difference between `@Controller` and `@RestController`?

`@Controller` is commonly used for returning views.
`@RestController` is commonly used for returning data directly.

## 5. What is Dependency Injection?

Dependency Injection means an object receives what it needs from outside instead of creating everything by itself.

## 6. Why is Dependency Injection useful?

It helps reduce tight coupling and makes code cleaner and easier to test.

## 7. What is a Spring bean?

A bean is a Java object managed by Spring.

## 8. What is the service layer used for?

The service layer usually contains business logic.

## 9. What does a repository do?

A repository helps the application work with stored data.

## 10. What does CRUD mean?

CRUD means Create, Read, Update, and Delete.

## 11. What is an entity?

An entity is a Java class that represents stored database data.

## 12. What is `JpaRepository`?

It is a Spring Data JPA interface that provides ready made data access methods.

## 13. What is DTO?

DTO means Data Transfer Object. It is used to control the data shape sent or received.

## 14. Why is validation important?

Validation helps stop bad or incomplete data from entering the application.

## 15. What is `ResponseEntity`?

`ResponseEntity` helps control the response body and status code.

## 16. What is the difference between authentication and authorization?

Authentication checks identity.
Authorization checks permission.

## 17. What is `application.properties` used for?

It is used for application configuration settings.

## 18. What is a profile in Spring Boot?

A profile is a group of settings for a specific environment such as development or production.

## 19. Why is logging important?

Logging helps developers understand application behavior and track problems.

## 20. What is Swagger or OpenAPI used for?

It is used to document APIs clearly and make them easier to understand and test.

## Interview tip

Do not try to sound too advanced if you do not fully understand a topic yet.
A clear and honest explanation is stronger than confusing jargon.
