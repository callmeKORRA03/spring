# Hello REST API Example

This is the first example project for Module 01.

## Purpose

This example helps beginners understand:

- how a Spring Boot Maven project looks
- where the main class lives
- how a controller is written
- how to test basic endpoints

## Project structure

- `pom.xml`
- `src/main/java/com/example/hellorestapi/HelloRestApiApplication.java`
- `src/main/java/com/example/hellorestapi/controller/HelloController.java`
- `src/main/resources/application.properties`

## Endpoints included

- `/hello`
- `/about`
- `/profile`

## Expected results

### `/hello`
Returns a simple text response.

### `/about`
Returns a short description.

### `/profile`
Returns simple JSON data.

## How to run

1. Open the project in NetBeans 2024
2. Allow Maven dependencies to load
3. Run the main application class
4. Open your browser and test the endpoints on port 8080

## Teaching note

This project is intentionally small.
It is made for beginner confidence, not complexity.
