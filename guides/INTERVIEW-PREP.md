# Spring Boot Interview Prep Guide

This guide helps you review the most likely beginner to junior Spring Boot topics after finishing the repository.

## Core questions you should be able to answer

### 1. What is Spring Boot?
Spring Boot is a framework tool built on Spring that helps developers build Java backend applications faster with less setup.

### 2. What is the difference between Spring and Spring Boot?
Spring is the broader framework. Spring Boot makes it easier and faster to start Spring applications with sensible defaults.

### 3. What is a controller?
A controller handles incoming web requests and returns responses.

### 4. What is the difference between `@Controller` and `@RestController`?
`@Controller` is commonly used for views, while `@RestController` is commonly used for returning data directly.

### 5. What is Dependency Injection?
Dependency Injection means a class receives what it needs from outside instead of creating everything by itself.

### 6. What is a bean?
A bean is a Java object managed by Spring.

### 7. What is the service layer?
The service layer contains business logic.

### 8. What does a repository do?
A repository helps the application work with stored data.

### 9. What is an entity?
An entity is a Java class that represents stored data in the database.

### 10. What does CRUD mean?
Create, Read, Update, Delete.

### 11. What is `ResponseEntity`?
It is a Spring class that helps control the response body and status code.

### 12. Why does validation matter?
Validation helps stop bad or incomplete input from entering the system.

### 13. What is a DTO?
A DTO is a Data Transfer Object used to send or receive only the data we want.

### 14. What is authentication?
Authentication checks who a user is.

### 15. What is authorization?
Authorization checks what a user is allowed to do.

## Good beginner practice answers

Try answering each question:

- in your own words
- in simple English
- with one example

## Interview habits that help

- explain clearly
- avoid overcomplicating simple concepts
- use examples
- say when you know the basics but are still growing in deeper areas
- stay honest and structured

You do not need to sound advanced to sound capable.
