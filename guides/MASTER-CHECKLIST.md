# Master Completion Checklist

Use this checklist to track your progress through the repository.

## Foundation stage

- [ ] I understand what backend development means
- [ ] I understand what Spring Boot is
- [ ] I understand HTTP request and response basics
- [ ] I understand Maven and `pom.xml`
- [ ] I understand Spring Boot project structure
- [ ] I can create and run a first endpoint

## Web development stage

- [ ] I can use `@RequestParam`
- [ ] I can use `@PathVariable`
- [ ] I can use `@PostMapping`
- [ ] I can receive JSON with `@RequestBody`
- [ ] I understand `ResponseEntity`
- [ ] I understand status codes
- [ ] I understand `@Controller` vs `@RestController`

## App structure stage

- [ ] I understand Inversion of Control
- [ ] I understand Dependency Injection
- [ ] I know what a bean is
- [ ] I understand the service layer
- [ ] I can use constructor injection
- [ ] I understand layered architecture

## Data and CRUD stage

- [ ] I understand JPA basics
- [ ] I can create an entity
- [ ] I can create a repository
- [ ] I understand CRUD operations clearly
- [ ] I can validate request data
- [ ] I can handle simple exceptions
- [ ] I understand DTO basics

## Quality stage

- [ ] I understand configuration properly
- [ ] I understand profiles
- [ ] I know why logging matters
- [ ] I understand testing basics
- [ ] I understand the role of Swagger or OpenAPI

## Industry readiness stage

- [ ] I understand public and protected endpoints
- [ ] I understand authentication
- [ ] I understand authorization
- [ ] I know how packaging works
- [ ] I understand basic deployment thinking
- [ ] I understand Docker at beginner level
- [ ] I have planned a capstone project

## Capstone stage

- [ ] I have chosen a capstone project idea
- [ ] I have listed my main entities
- [ ] I have designed my endpoints
- [ ] I have added validation rules
- [ ] I have added exception handling
- [ ] I have used DTOs
- [ ] I have tested the main flows
- [ ] I have documented the API

## Teaching readiness stage

- [ ] I can explain Spring Boot in simple English
- [ ] I can explain controller, service, and repository clearly
- [ ] I can explain CRUD with examples
- [ ] I can explain DTOs simply
- [ ] I can explain validation simply
- [ ] I can explain authentication and authorization clearly

## Final confidence check

- [ ] I can build a small Spring Boot API from scratch
- [ ] I can teach the fundamentals to beginners
- [ ] I can answer common junior Spring Boot interview questions
- [ ] I know what I should learn next after this repository
